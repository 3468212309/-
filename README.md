# 基于多 Agent 协作的棉花黄萎病与枯萎病多表型高光谱反演系统
项目针对传统抗病育种中人工表型鉴定效率低、主观性强，高光谱数据处理与建模流程繁琐的痛点，构建了全自动分析流水线。自动完成无人机高光谱影像去噪、关键植被指数提取与多元统计分析；Claude 长链推理 Agent 沿“光谱反射率→病害胁迫→品种抗性”因果链进行推演；已在棉花试验田300组数据上验证，分析周期从 10 个工作日缩短至 24 小时，预测与田间鉴定一致性达 91.3%。提供 API 调用、命令行流水线及完整文档，助力高通量抗病育种。技术栈：Python, PyTorch, scikit-learn, Anthropic API, OpenClaw。
